全局安装 docsify-cli 工具
```
npm i docsify-cli -g
```

如果想在项目的 ./docs 目录里写文档，直接通过 init 初始化项目。
```
docsify init ./docs
```

直接编辑 docs/README.md 就能更新文档内容
预览效果
```
docsify serve docs
```

浏览器输入
```
localhost:3000
```
